Airline Management System (Last Week)
=================================

How to run in VS Code / Terminal:

1. Open the AirlineManagementSystem folder.
2. Compile:
   javac -d out src/airlinemanagementsystem/*.java

3. Run:
   java -cp out airlinemanagementsystem.AirlineManagementSystem

Notes:
- Data is saved automatically in airline_data.ser.
- The first registered user becomes admin.
- This is a Java Swing/AWT file-based Airline Management System.
