Airline Management System - Week 3 Version

This is the third week progress version of the Java Airline Management System project.

Default Login:
Username: admin
Password: 123456

Week 3 Features:
1. Java Swing GUI continues from Week 2.
2. Login and registration system.
3. Flight management with add flight and flight table.
4. Ticket booking system.
5. Reservation table.
6. Payment system using booking ID.
7. Ticket cancellation system.
8. Seat decreases after booking and restores after cancellation.
9. File-based persistent data storage using Java Serialization.

Not included yet in Week 3:
- Staff Management
- Support Ticket
- Daily Report / Revenue Summary
- Final UI polish

These final features are planned for Week 4.

How to Run:
Double-click run.bat

Or use terminal:
javac -d out src\airlinemanagementsystem\*.java
java -cp out airlinemanagementsystem.AirlineManagementSystem
