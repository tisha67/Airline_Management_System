Airline Management System - Week 3 Version

This is the Week 3 progress version of the Airline Management System Java project.

Main Focus of Week 3:
- First GUI version using Java Swing
- Login window
- Registration window
- Main dashboard
- Flight management panel
- Add flight feature
- Flight list display using JTable
- File-based data storage continued from Week 2

Default Login:
Username: admin
Password: 123456

Features included:
1. User login and registration
2. Dashboard screen
3. Add flight information
4. Display all flights in a table
5. Save/load data using file handling

Features intentionally not included yet:
- Ticket booking
- Payment
- Cancellation
- Staff management
- Support ticket
- Daily report

These advanced features will be added in Week 4 and Week 5.

How to run:
1. Make sure JDK is installed.
2. Double click run.bat.
3. Or use command:
   javac -d out src\airlinemanagementsystem\*.java
   java -cp out airlinemanagementsystem.AirlineManagementSystem
