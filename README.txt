Airlines Management System - Week 2 Version

This is the second weekly progress version of the project.

Main focus of Week 2:
- Console based login system
- Registration system
- Basic file handling using serialization
- Save and load users
- Save and load passenger bookings
- Continue using basic OOP classes

Features included:
1. Register user
2. Login user
3. View sample flights
4. Add passenger booking
5. View saved passenger bookings
6. Data save/load using .dat files

Not included yet:
- GUI dashboard
- Flight add system
- Payment system
- Ticket cancellation
- Staff management
- Support ticket
- Daily report

How to run:
Double click run.bat

Or manually run:
javac -d out src\airlinemanagementsystem\*.java
java -cp out airlinemanagementsystem.AirlineManagementSystem
